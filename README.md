## Installation

### Prerequisites

Make sure you have the following installed before proceeding:

- **Node.js** (version >= 12.x): [Download here](https://nodejs.org/)
- **Expo CLI**: You can install it globally by running:

  ```bash
  npm install -g expo-cli

  npm install

  expo start

## Android Build

 expo run:android
 

## IOS Build

 expo run:ios
